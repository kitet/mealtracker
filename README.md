## Week 2 Project: Meal Tracker 

Author: [kitet](https://github.com/kitet)

A web application that let's user track meal they have consumed in a day and give details and analysis of calories consumed

## Keywords(Tech Used)
	angular 2, typescript, gulp, npm


## Running the web application
 
 1. Clone:
 ```
 git clone 
```
 2. Install dependencies:
 ```
 npm install and bower install
```
 3. Build the project:
 ```
 gulp build
```
